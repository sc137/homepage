ECIM 502 - INTERMEDIATE LIFE AND TECHNOLOGY INTEGRATION (EI)

Monday nights 6pm—7:50pm

➡️ View the Fall 2025 Booklet and register for classes.

Designed for the older adult, intermediate concepts are introduced related to current technologies to expand skill sets through the integration of technology into life. Topics include in depth concepts on Smartphone features for Android and Iphone, Ipads, Mac computers, Chromebooks, cloud computing, and security features.

This is an online synchronous class with all course requirements completed via Canvas. There are required live-stream meetings during all of the days/times that are on the course schedule.

The content in this course will span across multiple terms. Each semester will contain modules from the list below. Some modules can be thoroughly explored in a single week and some will require multiple weeks. The specific order for each semester will be listed in the syllabus.
